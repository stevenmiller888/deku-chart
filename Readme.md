
# deku-chart
[![NPM version][npm-image]][npm-url]
[![license][license-image]][license-url]

A chart component built with Deku and Chart.js.

<img src="https://cldup.com/L3mViZemfv.png" width="300" height="200">

## Installation

    $ npm install deku-chart

## Usage

```js
<Chart type="Bar">
  {
    {
      January:  [65, 28],
      February: [59, 48],
      March:    [80, 40],
      April:    [81, 19],
      May:      [56, 86],
      June:     [55, 27],
      July:     [40, 90]
    }
  }
</Chart>
```
